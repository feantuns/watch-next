## What it does

Watch Next lets you search for any movie or TV show, browse results, and instantly discover related titles — so you always have something great lined up.

- Search for a movie or series by name
- Browse results with poster images
- Click a title to see its details and a curated list of related movies
- Jump straight from one recommendation to the next

## Tech stack

- [React 18](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) for bundling and dev server
- [TailwindCSS](https://tailwindcss.com/) for styling
- [React Query](https://tanstack.com/query/v3) for data fetching and caching

## Getting started

```bash
npm install
npm start
```

The app will be available at `http://localhost:5173`.

## Build

```bash
npm run build
```
